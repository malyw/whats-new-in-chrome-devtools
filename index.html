Promises inspection and uncaught errors

Details

Now you can inspect all promises in your app

And also see uncaught promises errors and event have event listeners for them (check the details link) 

// Uncaught (in promise) ReferenceError
Promise.resolve()
    .then((res)=>{
        notDefined();
    });

// Caught promise error ReferenceError
Promise.resolve()
    .then((res)=>{
        notDefined();
    })
    .catch((err)=>{
        console.error('Catched promise error', err);
    });
